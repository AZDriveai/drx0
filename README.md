# DRX - منصة الذكاء الاصطناعي المتقدمة

## نظرة عامة

DRX هي منصة ذكاء اصطناعي متقدمة تدعم المحادثات التفاعلية مع إمكانيات إنشاء وتحرير المحتوى المختلف. تم بناء المنصة باستخدام Next.js 15 وتدعم عدة نماذج ذكاء اصطناعي.

## الميزات الرئيسية

### 🤖 نماذج الذكاء الاصطناعي المتعددة
- GPT-4o Mini & GPT-4o من OpenAI
- Claude 3 Haiku من Anthropic
- DeepSeek Chat للبرمجة والتحليل

### 💬 نظام المحادثة المتقدم
- محادثات تفاعلية في الوقت الفعلي
- دعم الرسائل متعددة الوسائط
- حفظ تاريخ المحادثات
- إعدادات الخصوصية (خاص/عام/غير مدرج)

### 🛠️ أدوات الإنتاجية
- **محرر الكود**: دعم متعدد اللغات مع تمييز الصيغة
- **محرر النصوص**: تحرير المستندات بتنسيق Markdown
- **محرر الجداول**: إنشاء وتحرير البيانات الجدولية
- **محرر الصور**: معالجة وتحرير الصور

### 🔐 نظام المصادقة
- تسجيل الدخول عبر Google
- وضع الضيف للاستكشاف
- إدارة الجلسات الآمنة

### 🎨 واجهة المستخدم
- تصميم متجاوب يدعم الأجهزة المختلفة
- دعم الوضع المظلم والفاتح
- واجهة عربية بالكامل
- شريط جانبي قابل للطي

## التقنيات المستخدمة

### Frontend
- **Next.js 15**: إطار العمل الأساسي
- **React 19**: مكتبة واجهة المستخدم
- **TypeScript**: لغة البرمجة المطبوعة
- **Tailwind CSS**: إطار عمل التصميم
- **Radix UI**: مكونات واجهة المستخدم
- **Framer Motion**: الرسوم المتحركة

### Backend
- **Next.js API Routes**: خدمات الخلفية
- **NextAuth.js**: نظام المصادقة
- **Drizzle ORM**: إدارة قاعدة البيانات
- **PostgreSQL**: قاعدة البيانات الرئيسية
- **Redis**: التخزين المؤقت (اختياري)

### AI & Tools
- **AI SDK**: تكامل نماذج الذكاء الاصطناعي
- **CodeMirror**: محرر الكود
- **ProseMirror**: محرر النصوص الغني
- **React Data Grid**: عرض وتحرير الجداول

## قاعدة البيانات

### الجداول الرئيسية

#### Users (المستخدمون)
- `id`: معرف فريد
- `email`: البريد الإلكتروني
- `name`: الاسم
- `image`: صورة الملف الشخصي
- `createdAt`, `updatedAt`: طوابع زمنية

#### Chats (المحادثات)
- `id`: معرف فريد
- `title`: عنوان المحادثة
- `userId`: معرف المستخدم
- `visibility`: مستوى الخصوصية
- `createdAt`, `updatedAt`: طوابع زمنية

#### Messages (الرسائل)
- `id`: معرف فريد
- `chatId`: معرف المحادثة
- `role`: دور المرسل (user/assistant/system)
- `content`: محتوى الرسالة
- `metadata`: بيانات إضافية
- `createdAt`: طابع زمني

#### Artifacts (المصنوعات)
- `id`: معرف فريد
- `messageId`: معرف الرسالة
- `type`: نوع المصنوع (code/text/image/sheet)
- `title`: العنوان
- `content`: المحتوى
- `language`: اللغة (للكود)
- `createdAt`, `updatedAt`: طوابع زمنية

## التثبيت والتشغيل

### المتطلبات
- Node.js 18+
- PostgreSQL
- Redis (اختياري)

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd drx
```

2. **تثبيت التبعيات**
```bash
pnpm install
```

3. **إعداد متغيرات البيئة**
```bash
cp .env.example .env.local
```

4. **تحديث متغيرات البيئة**
```env
# Database
POSTGRES_URL=postgresql://username:password@localhost:5432/drx_db

# Authentication
NEXTAUTH_SECRET=your-secret-key-here
NEXTAUTH_URL=http://localhost:3000
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret

# AI Models
OPENAI_API_KEY=your-openai-api-key
ANTHROPIC_API_KEY=your-anthropic-api-key
XAI_API_KEY=your-xai-api-key

# Redis (optional)
REDIS_URL=redis://localhost:6379
```

5. **إعداد قاعدة البيانات**
```bash
pnpm db:generate
pnpm db:migrate
```

6. **تشغيل المشروع**
```bash
pnpm dev
```
